A modern, responsive landing page for a hospital website.

Built with:
- **Gatsby.js** (React Framework)
- **Figma** (UI/UX Design)
- **REST API** + **Axios** (for dynamic reviews)
- **Bootstrap** (responsive design)
- **Vercel** (deployment)
- **Git** (version control)
